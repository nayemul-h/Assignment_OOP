→ int id

→String name

→int[] marks (to store marks of 3 subjects)

→float average

→A constructor that takes id, name, and marks as input and stores them.
Inside the Student class, create two methods:

→calculateAverage(): calculates the average of the 3 subject marks.

→printDetails(): prints the ID, name, marks, and average of the student.
In the main() method:

→Create 3 student objects manually.

→Take input for each student (ID, name, and marks for 3 subjects).

→Call calculateAverage() and printDetails() for each student.
